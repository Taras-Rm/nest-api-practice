# Users API (client)

This is basic client for users API testing.

## Technologies

- React.js
- Antd
- Axios

## Run app

1. Clone repository:
``` bash
git clone https://github.com/Taras-Rm/express-api-practice.git
```

2. Move to client folder:
``` bash
cd ./client
```

3. Install dependencies
``` bash
npm install
```

4. Start app
``` bash
npm run dev
```
